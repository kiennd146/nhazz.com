This application integrates the comments system component JComments into SobiPro component.

To display the comments and the comment form within the details view, add following code into your details view template entry/details.xsl: 
        <xsl:value-of select="jcomments" disable-output-escaping="yes" />
To show the number of comments for an entry within the vCard, add this code into your vCard template common/vcard.xsl: 
        <xsl:value-of select="jcomments"/>
This app does not requires any additional configuration within the SobiPro component.
